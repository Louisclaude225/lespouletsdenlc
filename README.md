# Mon Application de Commande

Application web permettant aux clients de consulter des produits et passer des commandes.

## Fonctionnalités

- Consultation des produits
- Ajout de produits au panier
- Passage de commande
- Interface adaptée aux téléphones et ordinateurs

## Mise en ligne

Cette application est publiée avec GitHub Pages.

## Fichiers principaux

- `index.html` : page principale de l’application
- `README.md` : présentation du projet

## Utilisation

Ouvrez le lien GitHub Pages du projet dans un navigateur pour utiliser l’application.
